# Solo Gainz

A premium, immersive fitness and combat gaming hybrid built with Flutter.

## Overview
Solo Gainz combines the discipline of daily workout tracking with the excitement of a combat-based dungeon crawler. Users progress through quests, hone their skills in a training simulator, and conquer dungeons to build their strength.

## Key Features
*   **Immersive UI**: High-fidelity design with glassmorphism, dynamic gradients, and smooth animations.
*   **Combat Engine**: Frame-by-frame physics and hitbox detection for a responsive gaming experience.
*   **Training Mode**: A specialized environment to practice combos and test physics.
*   **Quest System**: Daily and custom missions to track fitness progress.
*   **Dungeons**: Progress-based challenges with distinct visual themes.

## Platform Support
*   Android
*   iOS
